Implement Defogging with Python


Features
--------

* Defogging single foggy image

install
-------

Use pip to install

.. code-block:: bash::

    $ pip install defogging

usage
-----

Defogging a single foggy image

.. code-block:: bash::

    $ defogging your_img.bmp

the result will be saved as your_img_defogging.bmp
